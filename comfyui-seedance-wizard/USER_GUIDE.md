# Seedance Wizard 使用指南

## 简介

Seedance Wizard 是一个集成在 ComfyUI 中的 AI 视频生成工具。它通过 AI 分析您的图片和创意想法，自动生成高质量的 AI 视频。

---

## 快速开始（3 步）

### 1. 获取 API Key

打开 Seedance Wizard 设置面板：

1. 在 ComfyUI 右侧面板找到 **Seedance Wizard** 标签页
2. 点击右上角 **⚙ 设置** 按钮
3. 选择 **注册** 或 **登录** 标签：
   - **新用户**：输入邮箱 + 密码（至少 8 位），点击「注册并激活」，系统自动创建账号并生成 API Key
   - **已有账号**：输入邮箱 + 密码，点击「登录并激活」，自动获取 API Key
4. 保存后即可使用

> 也可以在 [see4dance.com](https://see4dance.com) 网页端注册和管理 API Key。

### 2. 选择使用模式

Seedance Wizard 支持 **两种工作模式**：

| 模式 | 适用场景 | 特点 |
|------|---------|------|
| **侧边栏模式** 🎨 | 手动交互、反复调参 | 可视化界面、实时预览、所见即所得 |
| **管线模式** 🔗 | 自动化工作流、批量处理 | 节点连接、无需手动操作、可串联上下游 |

### 3. 开始生成

---

## 模式一：侧边栏模式（交互式向导）

这是最直观的使用方式。在右侧面板中按照 4 个步骤操作：

### Step 1 — 上传图片 & 输入想法

- **上传图片**：点击虚线框上传，或直接将图片拖入。支持 PNG / JPG / WEBP
- **输入创意想法**：用一两句话描述你想要的效果

> 例如："把这个产品做成有质感的商业广告，突出高端感和科技感"

- **选择视频比例**：16:9 横屏 / 9:16 竖屏 / 1:1 方形
- 点击 **✨ 开始 AI 分析**

### Step 2 — AI 分析

系统自动进行三步分析：
1. **Qwen VL** 识别图片内容
2. **DeepSeek** 制定创作方案
3. **通义万象** 生成预览图

等待几秒钟，AI 会根据您的图片和想法自动配置最佳参数。

### Step 3 — 参数调优

进入参数面板后，您可以微调 AI 给出的方案：

**风格参数**（下拉选择）：
- **画面风格**：电影感 / 商业广告 / 纪录片 / 社交媒体 / 艺术创意
- **灯光效果**：明亮日光 / 黄金时段 / 柔和漫射 / 戏剧阴影 / 霓虹夜景
- **构图景别**：特写 / 中景 / 广角 / 俯瞰 / 仰视
- **情绪氛围**：活力动感 / 宁静优雅 / 神秘感 / 欢快喜悦 / 戏剧张力
- **色调倾向**：暖色调 / 冷色调 / 鲜艳饱和 / 低饱和 / 单色系

**精细调节**（滑块）：
- **动感强度**：控制画面中物体运动幅度
- **景深虚化**：背景模糊程度
- **细节丰富**：画面细节密度
- **饱和度**：色彩鲜艳程度

每次调整参数后，点击 **↺ 重新生成预览** 查看效果。满意后点击 **▶ 生成视频**。

### Step 4 — 等待生成 & 获取结果

- 视频生成大约需要 3-10 分钟（取决于服务器负载）
- 完成后视频文件自动保存到 `ComfyUI/output/seedance/` 目录
- 可以复制视频链接或点击「重新制作」开始新的创作

## 模式二：管线模式（节点图工作流）

管线模式让 Seedance Wizard 真正融入 ComfyUI 的节点工作流——您可以像使用 KSampler 一样使用它。

### 核心原理

管线模式下，节点接收来自上游节点的 **IMAGE**（图片）和 **STRING**（文字）输入，自动完成：**AI 分析 → 生成预览 → 生成视频 → 下载结果**，并将结果输出给下游节点。

### 工作流示例

```
┌───────────┐  IMAGE    ┌────────────────────┐
│ LoadImage │──────────►│                    │
└───────────┘           │                    │  video_path ┌───────────┐
                        │ SeedanceWizardNode │────────────►│ VideoSaver│
┌───────────┐  STRING   │                    │            └───────────┘
│  TextBox  │──────────►│                    │
└───────────┘           │                    │  preview   ┌────────────┐
                        │                    │───────────►│ PreviewImg │
┌───────────┐  STRING   │                    │            └────────────┘
│ StyleSelect──────────►│                    │
└───────────┘           └────────────────────┘
```

### 搭建管线工作流（分步说明）

#### ① 添加节点

在节点图中右键 → `Add Node` → `Seedance Wizard`，可用的节点有：

| 节点 | 用途 |
|------|------|
| **Seedance Wizard** | 主节点，连接 IMAGE + 文字输入，输出视频和预览图 |
| **Seedance API Key** | （可选）显式传入 API Key，适合分享工作流时避免泄露 |
| **Seedance Image Input** | （可选）标记图片输入，让工作流更清晰可读 |

#### ② 连接上游输入

**必需的连接**：
- 将 `LoadImage` 的输出连接到 `Seedance Wizard` 的 `image` 输入

**可选的连接**：
- 将文字节点的输出连接到 `prompt` 输入（创意想法描述）
- `style`、`mood`、`camera`、`lighting`、`color_tone` 等可以用下拉菜单选择，也可以用其他节点的 STRING 输出来控制

#### ③ 连接下游输出

- `video_path` (STRING) — 生成视频的本地文件路径，可接 `VideoSaver` 或自定义处理节点
- `preview_thumbnail` (IMAGE) — AI 生成的预览图（ComfyUI tensor 格式），可直接显示或进一步处理
- `final_prompt` (STRING) — 最终发送给视频模型的英文 Prompt，可用于记录或调试

#### ④ 设置参数 & 执行

- 通过节点的下拉菜单设置 **画面比例、风格、氛围、镜头、灯光、色调**
- 调整 **动感强度**（0~1）、**Seed**（0 为随机）
- 点击 **Queue Prompt** 开始执行
- 节点会自动完成全流程，您可以在 ComfyUI 的终端日志中看到进度

### 管线模式输入参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `trigger` | INT | 侧边栏模式用，管线模式忽略 |
| `image` | IMAGE | **管线模式必需**：接入任意图片节点的输出 |
| `prompt` | STRING | 创意想法，英文效果更好，2-3 句话 |
| `aspect_ratio` | 下拉 | 视频比例：16:9 / 9:16 / 1:1 |
| `style` | 下拉 | 画面风格：cinematic / commercial / documentary / social_media / artistic |
| `mood` | 下拉 | 情绪氛围：energetic / serene / mysterious / joyful / dramatic |
| `camera` | 下拉 | 镜头构图：close_up / medium_shot / wide_shot / aerial_view / low_angle |
| `lighting` | 下拉 | 灯光效果：bright_daylight / golden_hour / soft_diffused / dramatic_shadows / neon_night |
| `color_tone` | 下拉 | 色调倾向：warm / cool / vibrant / muted / monochrome |
| `motion_intensity` | FLOAT | 动感强度：0.0（静止）~ 1.0（大幅运动） |
| `seed` | INT | 随机种子，0 = 随机 |
| `api_key` | STRING | （可选）覆盖配置文件中的 API Key |

### 管线模式输出说明

| 输出 | 类型 | 说明 |
|------|------|------|
| `video_path` | STRING | 生成的 MP4 视频本地路径 |
| `preview_thumbnail` | IMAGE | AI 生成的预览图（可在 ComfyUI 中直接显示） |
| `final_prompt` | STRING | 最终合并后的英文 Prompt |

## 双模式如何共存

同一个 `SeedanceWizardNode` 同时支持两种模式，节点自动检测：

- **当 `image` 输入被连接且有数据** → 自动进入**管线模式**（使用节点图中的数据）
- **当 `image` 输入为空** → 自动进入**侧边栏模式**（等待用户在向导面板中操作）

两种模式互不干扰。您可以先在侧边栏中交互式地调试参数，确认效果后再切换到管线模式进行批量处理。

## 高级技巧

### 批量生成

将多个 LoadImage 分别连接到不同的 SeedanceWizardNode，ComfyUI 会依次处理每个节点。

### 自动化工作流

管线模式可以串联在更大的工作流中。例如：

```
LoadImage → SeedanceWizardNode → VideoSaver
                                → UpscaleModel → EnhancedVideoSaver
```

## 常见问题

**Q: 侧边栏模式和管线模式能同时用吗？**
A: 可以。同一个节点在侧边栏中操作数据不会影响管线模式的节点。它们是独立实例。

**Q: 管线模式下怎么看到进度？**
A: 侧边栏面板会自动显示管线执行的实时进度（分析 → 预览 → 生成），视频进度条会实时更新。

**Q: API Key 写在哪里？**
A: 侧边栏设置面板中保存后，自动写入 `ComfyUI/user/default/seedance_config.json`。管线模式下也可以用 `SeedanceAPIKeyNode` 显式传入。

**Q: 预览图没生成怎么办？**
A: 预览图依赖后端预览服务。即使预览失败，视频仍可以正常生成。预览图为空时会返回 None。

**Q: 视频生成失败？**
A: 检查 API Key 是否有效、余额是否充足。在侧边栏设置面板中查看状态指示灯。

**Q: 管线模式和侧边栏模式的 trigger 参数是干什么的？**
A: trigger 仅用于侧边栏模式。当用户在侧边栏中提交任务后，将 trigger 从 0 改为 1，节点开始轮询等待视频完成。
