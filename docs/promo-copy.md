## 🧵 Medium (Reddit, Hacker News, Indie Hackers)

**Title: I built an AI video workshop that costs the same as the raw API — no code, no subscription**

I got tired of paying $15–20/month for AI video tools that don't even include model API costs. So I built Seedance Studio.

**What it does:**
Upload a reference image → AI analyzes style/mood/camera → generates preview → video draft → AI music → final render. All in one browser tab.

**The models:**
18 models built in — Seedance 2.0 (ByteDance), Kling O3 (Kuaishou), Veo 3.1 (Google), Sora 2, Wan 2.7, Suno V5, Midjourney V7, and more. Pick whichever fits your style. Models consume different credits based on their API cost — cheaper models cost fewer credits, premium ones cost more. You choose.

**The pricing:**
- Starter $11.9 = 120 credits ($0.20/sec) — 6 videos
- Standard $23.9 = 270 credits ($0.16/sec) — 15 videos
- Pro $69.9 = 900 credits ($0.14/sec) — 50 videos

Pay once. Credits never expire. No monthly lock-in.

**How it compares:**
Raw Kling API is $0.08/sec. Raw Seedance 2.0 API is $0.14/sec. Our Pro plan is $0.14/sec — but you get a full UI, AI scriptwriting, background music, and 18 models to switch between. Same price, more value.

**Why I built it this way:**
Monthly subscriptions suck for creators who don't produce videos every day. With credits that never expire, you buy when you need and use when you want. The discount ladder (20% off Standard, 30% off Pro) makes it cheaper the more you buy.

**Stack:** Python FastAPI + PostgreSQL + Railway + Cloudflare R2 + Creem payments

Try it: https://see4dance.com
GitHub: https://github.com/liuhongyi0/seedance-transfer

Happy to answer questions.
